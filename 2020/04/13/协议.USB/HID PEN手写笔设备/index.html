HID PEN手写笔设备

描述符

报文说明

Pen Input报文共9Byte

Byte0 Byte1 Byte2 Byte3 Byte4 Byte5 Byte6 Byte7 Byte8 Byte9 Byte10 Byte11

Byte0 -- 开关
      |--Bit0: Tip switch触点开关
      |--Bit1:Barrel Switch桶开关
      |--Bit2:Eraser Switch擦开关
      |--Bit3:Invert反转
      |--Bit4:In Range
      |--Bit5:填0，保留
      |--Bit6:填0，保留
      |--Bit7:填0，保留

    笔操作流程：
        笔尖悬空 0x10
        笔尖接触屏幕 0x11
        笔尖抬起 0x10
        笔尖释放 0x00  

    橡皮擦操作流程：
        橡皮擦悬空 0x1a
        橡皮擦接触屏幕 0x16
        橡皮擦抬起 0x1a
        橡皮擦释放 0x00

Byte 1:0x00保留

Byte 2~3 : X坐标

Byte 4~5 : Y坐标

Byte 6~7 : Tip Pressure 压力 

Byte 8~9 : X Tilt

Byte 10~11 :Y Tilt

Byte8~11可以在HID描述符里删除
